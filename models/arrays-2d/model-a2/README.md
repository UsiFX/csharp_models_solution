# Model A2

## Problem
Print elements that are either:
Greater than 25 and divisible by 2
Less than 25 and divisible by 3

## Input
22 16 90
37 18 50

## Output
Elements matching the conditions.

## Logic
Nested loops to traverse array and check conditions.

## How to Run

```bash
make run topic=arrays-2d model=a2
```
