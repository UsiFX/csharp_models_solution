# Model D1

## Problem
Print array elements in reverse order only if they are greater than 20.

## Input
50 45 40 35 30 25 20 15 10

## Output
50 45 40 35 30 25

## Logic
Loop the array in reverse and print elements greater than 20.

## How to Run

```bash
make run topic=arrays-1d model=d1
```
