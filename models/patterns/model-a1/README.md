# Model A1

## Problem
Print the following pattern using a nested for loop:

``` bash
9 8 7 6
5 4 3
2 1
0
```

## Input
No input.

## Output
```bash
9 8 7 6
5 4 3
2 1
0
```

## Logic
The outer loop controls rows and the inner loop prints decreasing numbers per row.

## How to Run

```bash
make run topic=patterns model=a1
```
