# Model B7

## Problem
Count occurrences of a number X in an 8-element array.

## Input
- Array of 8 integers
- A number X to search for

## Output
Number of times X appears in the array.

## Logic
1. Read 8 integers into an array
2. Read the number X to search for
3. Initialize a counter to 0
4. Iterate through each element in the array:
   - If element equals X, increment the counter
5. Print the final count

## How to Run

```bash
make run topic=arrays-1d model=b7
```
