# Model C2

## Problem
Print numbers within [25,75] from a larger array.

## Input
Array of integers (e.g., 44 4 18 2 8 64 32 3 47 50 22 12)

## Output
Elements that fall within the range [25, 75] inclusive (e.g., 44 64 32 47 50)

## Logic
1. Read the array of integers
2. Iterate through each element:
   - Check if element >= 25 AND element <= 75
   - If true, print the element
3. Print all qualifying elements in one line

## How to Run

```bash
make run topic=arrays-1d model=c2
```
